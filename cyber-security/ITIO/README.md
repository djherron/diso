# ITIO

## Overview 

**ITIO: Insider Threat Indicator Ontology**

A cybersecurity ontology with a focus on the subdomain of **insider threat indicators**

ITIO website
* https://www.sei.cmu.edu/library/an-insider-threat-indicator-ontology/
