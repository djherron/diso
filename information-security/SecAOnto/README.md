# SecAOnto

## Overview

**SecAOnto: Security Assessment Ontology**

paper:
* `Towards an Ontology of Security Assessment - A Core Model Proposal`, Ferrucio de Franco Rosa, Mario Jino, and Rodrigo Bonacin, 2018

SecAOnto is an **information security** ontology whose focus is on **security assessment**. It defines **security assessment** as the intersection between **information security** and **software assessment**. The authors see **security assessment** as dealing with "concepts, processes, methods and techniques crucial to the development and maintenance of secure systems".
