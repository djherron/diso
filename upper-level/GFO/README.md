# GFO

## Overview

**GFO: General Formal Ontology**

GFO website
https://www.onto-med.de/ontologies/gfo
> GFO includes elaborations of categories like objects, processes, time and space, properties, relations, roles, functions, facts, and situations

There are 2 OWL versions of GFO:
* the **stable core** ontology: `gfo-basic.owl`
* an **extended** version: `gfo.owl`

`gfo.owl`
* http://www.onto-med.de/ontologies/gfo.owl
* version 1.0, build 9, 28.08.2006

`gfo-basic.owl`
* http://www.onto-med.de/ontologies/gfo-basic.owl
* version 1.0, build 13, 07.10.2008

There is also a tiny variation of `gfo-basic.owl` called `gfo-basic-renamed.owl`. 
> ... a modified version of `gfo-basic.owl`, because some users have reported problems with the use of the GFO namespace, http://www.onto-med.de/ontologies/gfo, as base namespace for gfo-basic
* https://www.onto-med.de/sites/www.onto-med.de/files/files/uploads/Ontologies/gfo-basic-renamed.owl
