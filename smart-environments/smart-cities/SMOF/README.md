# SMOF

## Overview 

**SMOF: Smart City Ontology Framework**

paper:
* https://www.mdpi.com/2624-6511/8/5/165
* `Smart City Ontology Framework for Urban Data Integration and Application (He, 2025).pdf`
> A hierarchical ontology (SMOF) with universal and extended properties and a concise relation scheme that draws on authoritative standards/ontologies (e.g., IFC, CityGML, SSN/SOSA) to support city-wide, cross-domain data integration

SMOF ontology download:
* https://figshare.com/articles/dataset/SMOF_IJGIS_/24996683
* see `SMOF.rdf`

Using Protege, we converted `SMOF.rdf` to `SMOF.ttl`
