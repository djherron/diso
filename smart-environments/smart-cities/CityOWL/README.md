# CityOWL

## Overview 

**CityOWL: an OWL rendering of the CityGML UML+XML schema**

CityOWL is an OWL rendering of CityGML (UML+XML schema)
* https://www.ogc.org/standards/citygml/

CityOWL is a network of ontologies for CityGML 3.0, transformed from the official UML model.

websites:
* https://dataset-dl.liris.cnrs.fr/rdf-owl-urban-data-ontologies/Ontologies/Readme.html
* https://dataset-dl.liris.cnrs.fr/rdf-owl-urban-data-ontologies/Ontologies/CityOWL/CWA/3.0/
  
The master ontology, `CityOWL.ttl`, imports all of the component ontologies. The component ontologies, in turn, do some importing as well.

We created a merged version of CityOWL using Protege.
* see `CityOWLmerged.ttl`
