# BRICK

## Overview 

**BRICK: a buildings ontology**

website
* https://brickschema.org/
> Brick is an open-source effort to standardize semantic descriptions of the physical, logical and virtual assets in buildings and the relationships between them.

GitHub repo
* https://github.com/BrickSchema/Brick/
* the repo has alignments to other ontologies, like BOT
  - but the alignment is small; just 9 mappings

`Brick+imports.ttl` (7.5 MB)
* recommended for end-applications
* contains the Brick and RealEstateCore ontologies, as well as all imports; no other dependencies are required

docs
* https://brickschema.readthedocs.io/en/latest/index.html
